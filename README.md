## Requirements

* .Net 5.0
* Your favorite IDE (Visual studio is probably best but VS Code with .Net CLI works fine as well)

## App Settings

* Create an `appsettings.json`, copy the fields from `appsettings.example.json`, and finally fill out the fields.
